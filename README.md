# 직원 관리 시스템 개발 예시

## 요약

Oracle DB 의 예시 테이블을 이용하여 직원 관리 시스템을 개발한 예시입니다. 

## 프로젝트 기능 

- 직원 정보 조회
- 직원 이력 정보 조회
- 부서 정보 조회 
- 위치 정보 조회
- 부서 급여 인상
- 직원 정보 수정
- 강좌 정보 검색

## 사용하는 기술

- Java 17
- Spring Boot 
- Spring Boot Docker Compose
- Spring Data JPA
- QueryDSL
- Spring Security
- JUnit 5
- MySQL
- Docker
- Docker Compose

## 프로젝트 실행 방법

1. 프로젝트를 클론합니다.

```shell
git clone https://github.com/pjc1991/employee-management-system.git &&
cd employee-management-system
```

2. 프로젝트를 빌드합니다.

```shell
./gradlew clean build
```

3. 어플리케이션을 실행합니다.
    
```shell
./gradlew bootRun
```
